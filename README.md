## Installing and Running the App
### 1️⃣ Clone Repository
```
git clone https://github.com/imamtashbir95/fastrans-express.git
cd fastrans-express
```

### 2️⃣ Install Dependencies
```
npm install
```

### 3️⃣ Run App
Development mode:
```
npm run dev
```
Production mode:
```
npm start
````
App will run on http://localhost:3000/ 🚀
